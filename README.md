# Local RAG Ollama

Local RAG chatbot using Ollama, LangChain, and LLaMA3 for private AI document querying.

## Features
- PDF document ingestion
- Local embeddings
- Offline AI querying
- Privacy-focused
- No OpenAI API required

## Tech Stack
- Python
- Ollama
- LangChain
- ChromaDB
- LLaMA3

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```
